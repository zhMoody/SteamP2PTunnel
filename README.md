# Steam P2P Tunnel

**Steam P2P Tunnel** 是一款利用 Steam P2P 网络，将您本地的游戏伪装成一个局域网（LAN）游戏，让您和您的 Steam
好友能够轻松联机的工具。它为许多不支持在线联机但支持局域网联机的游戏提供了一个简单、安全、低延迟的解决方案。

## 技术说明 (Technical Explanation)

本部分为对技术感兴趣的用户和开发者提供，解释了 Steam P2P Tunnel 的工作原理。

### 核心原理

Steam P2P Tunnel 的核心是利用 **Steamworks SDK** 提供的 **P2P 网络功能**。Steam 的 P2P 网络不仅仅是简单的两台电脑直连，它还包含了
Valve 功能强大的全球中继网络（**Steam Datagram Relay, SDR**）。

这意味着：

* **NAT 穿透**: 您和您的朋友无需进行复杂的路由器端口转发或设置。Steam 的网络会自动为您找到最佳连接路径。
* **低延迟**: Steam 会在全球部署的服务器中选择离您和朋友最近的中继点，以优化网络延迟。
* **安全性**: 所有通过 Steam P2P 网络传输的数据都经过加密，保证了通信的私密性。

### 技术栈

* **应用框架**: **Tauri (v2)** - 一个使用 Web 技术构建桌面应用的框架。它将 Rust 后端与 Web 前端捆绑在一起，实现了高性能、高安全性和极小的打包体积。
* **后端逻辑**: **Rust** - 负责与 Steamworks SDK 交互、处理 P2P 网络连接、以及在本地TCP端口和P2P网络之间转发数据。
* **前端界面**: **React + TypeScript + Vite** - 提供了现代、美观且响应迅速的用户界面。

### 工作流程

1. **启动与认证**:

* 应用启动时，Rust 后端会使用测试 App ID `480` (Spacewar) 调用 `Client::init_app(480)` 来初始化并连接到本地正在运行的
  Steam 客户端。

2. **创建/加入房间 (Lobby)**:

* **房主**: 创建一个 Steam Lobby。这本质上是在 Steam 的匹配服务器上注册了一个“房间”，并获得一个唯一的 Lobby ID。
* **客户端**: 通过房主分享的 Lobby ID 加入这个房间。

3. **建立 P2P 连接**:

* **客户端**: 在成功加入 Lobby 后，获取房主的 Steam ID，并调用 `connect_p2p` 向房主发起 P2P 连接请求。
* **房主**: 通过 `ListenSocket` 监听来自客户端的 P2P 连接请求并接受。

4. **数据隧道 (Data Tunneling)**:

* 连接建立后，一个双向的数据隧道即告形成。
* **（发送方）**: 本地游戏客户端的数据发送到 Steam P2P Tunnel 监听的本地 TCP 端口 -> Steam P2P Tunnel 将 TCP 数据流打包成自定义的
  `TunnelPacket` -> 通过 `send_message` 经由 Steam P2P 网络发送给对方。
* **（接收方）**: Steam P2P Tunnel 通过 `receive_messages` 从 P2P 网络接收 `TunnelPacket` -> 解析数据包 -> 将数据通过 TCP
  连接转发到对方本地游戏服务器监听的端口。
* 这个过程是双向的，从而实现了对局域网游戏流量的完全模拟。

## 使用说明 (User Guide)

请按照以下步骤与您的 Steam 好友进行联机。

### **准备工作 (Prerequisites)**

* **您和您的朋友**都需要在电脑上安装并登录 **Steam 客户端**。
* 您和您的朋友在 Steam 上必须是**好友关系**。
* **您和您的朋友**都需要下载并运行 **Steam P2P Tunnel** 应用程序。
* **关闭 Windows 防火墙**或允许 Steam P2P Tunnel 通过防火墙，以避免潜在的网络连接问题。

### **【房主】要做的事 (Host's Steps)**

1. **启动**: 先确保 Steam 正在运行，然后打开 `Steam P2P Tunnel.exe`。
2. **配置端口**:

* 在 "本地游戏端口" 输入框中，填入您游戏用于局域网联机的端口号。
* *例如：Minecraft Java 版通常是 `25565`。您需要在游戏中“对局域网开放”，游戏会告诉您一个端口号，把那个端口号填在这里。*

3. **创建房间**:

* 确保 "我是房主" 标签页被选中。
* 点击 **"启动房间"** 按钮。

4. **分享ID**:

* 成功后，界面会显示 "正在主持"。点击右上角的 **"复制ID"** 按钮。
* 将这个复制好的ID（一长串数字）通过聊天软件发送给您的朋友。

5. **开始游戏**:

* 进入游戏，创建世界，并选择“对局域网开放”。
* 现在，等待您的朋友加入即可。

### **【加入方/朋友】要做的事 (Player's Steps)**

1. **启动**: 同样，先确保 Steam 正在运行，然后打开 `Steam P2P Tunnel.exe`。
2. **配置端口**:

* 在 "本地游戏端口" 输入框中，填入**和房主完全相同**的端口号。
* *这非常重要，它告诉 Steam P2P Tunnel 应该把数据转发到您本地游戏的哪个“门牌号”。*

3. **加入房间**:

* 点击切换到 "加入游戏" 标签页。
* 将房主发给您的房间ID粘贴到 "房间 ID" 输入框中。

4. **连接**:

* 点击 **"立即加入"** 按钮。
* 成功后，界面会显示 "已连接"。

5. **开始游戏**:

* 现在，进入您的游戏。
* 在游戏的多人游戏/联机菜单中，选择“直接连接”或“添加服务器”。
* 在服务器地址栏中，输入 `127.0.0.1:<端口号>`。
* *例如：如果端口是 `25565`，您就应该输入 `127.0.0.1:25565`。*
* 点击连接，您现在应该能成功加入房主的游戏了！

### **断开连接**

当游戏结束后，任何一方都可以点击 Steam P2P Tunnel 界面上的 **"关闭房间"** 或 **"断开连接"** 按钮来安全地关闭 P2P 隧道。
